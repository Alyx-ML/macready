# MacReady Frontend

React frontend for the MacReady web app.

## Development

Run from the repository root:

```bash
bun run dev
```

This starts the Bun API server on port `8421` and the Vite frontend on port `5173`.

## Useful Commands

From `frontend/`:

```bash
bun run dev
bun run build
bun run typecheck
bun run lint
```
